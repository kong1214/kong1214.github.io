metam

Flask, SQLAlchemy, React, Redux, Redux, HTML5, CSS3, Javascript, Render

Metam is a project management site, where a logged in user can create projects as well as sections and tasks associated with those projects. Each project can also have a team of other Metam users, to allow for easy and seamless collaboration. Task assignments, a project audit log, and user profiles are currently in the works to be implemented.

RiftPay

Flask, SQLAlchemy, React, Redux, Redux, HTML5, CSS3, Javascript, Render

Riftpay is a Splitwise clone. Worked on by a team of four engineers, including myself, it allows users to make an expense with multiple people and split that transaction between those associated users in varying ways. Users involved in these transactions can also comment on the transactions and see the comments of other users. Users can also add other users as friends and see what their balance is with each friend.

airana

Express, Sequelize, React, Redux, Redux, HTML5, CSS3, Javascript, Render

Airana is an airbnb clone. Users can create and view listings. A logged in user can leave and edit their own reviews for each listing. Users can also create non-conflicting reservations of these listings. The next feature to be implemented is a Google Maps API that would also allow users to search specific areas and examine listings on an easy to navigate interface.
